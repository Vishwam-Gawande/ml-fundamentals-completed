# 🧠 AI News Assistant

An AI-powered web application that delivers real-time news with instant summaries using NLP models.

---

## 🚀 Live Demo
👉 https://ai-news-app-4e8x.onrender.com

---

## 📌 Features

- 📰 Fetch real-time news (Technology, AI, World)
- 🧠 AI-powered summarization
- ⚡ Fast and responsive UI
- 🔄 Dynamic updates using JavaScript
- ❗ Error handling for API failures
- 🌐 Deployed and accessible online

---

## 🛠️ Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- APIs: NewsAPI
- AI: Hugging Face Transformers (Inference API)
- Deployment: Render

---

## 🧠 How It Works

1. User selects a news category  
2. App fetches articles using NewsAPI  
3. User clicks "Summarize"  
4. Text is sent to AI model  
5. Model generates summary using NLP  

---

## 🔍 AI Pipeline

User Input  
→ Text Processing  
→ Embeddings (text → numbers)  
→ Model Inference  
→ Summary Output
